# Lodestar

AI-Powered Stock Screener & Active Investing Companion.

Combines fundamental quality filters with technical moving average signals (MA150/MA200), enriched with AI-generated investment rationale, target prices, stop losses, and a live recommendation lifecycle that updates daily.

In development — see [`docs/superpowers/specs/`](docs/superpowers/specs/) for the current design spec.
